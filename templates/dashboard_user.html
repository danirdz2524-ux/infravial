Manual de Usuario - InfraVial
📌 ¿Cómo crear una nueva ficha?
  1. Haz clic en "Nueva Ficha" en el menú lateral.
  2. Completa todos los campos del formulario.
  3. Puedes subir una foto arrastrándola o haciendo clic en "Subir Foto".
  4. Presiona Enter para moverte al siguiente campo rápidamente.
  5. Haz clic en "Guardar Ficha" para guardar.
📋 ¿Cómo ver y gestionar registros?
  1. Ve a "Ver Registros" en el menú.
  2. Usa el buscador para filtrar por provincia, cantón o número de ficha.
  3. Usa los botones ✏️ Editar y 🗑️ Eliminar.
📤 ¿Cómo exportar datos a CSV o Excel?
  1. En la página de Registros, haz clic en "Exportar Datos" (botón verde).
  2. Elige entre formato CSV (compatible con QGIS) o Excel.
  3. También puedes descargar una plantilla QGIS de ejemplo.
🗺️ ¿Cómo importar a QGIS?
  1. Abre QGIS → Capa → Añadir capa → Añadir capa de texto delimitado.
  2. Selecciona el archivo CSV exportado.
  3. Separador: Coma (,), Codificación: UTF-8.
  4. Para crear puntos: usa COORD_X_ESTE como X y COORD_Y_NORTE como Y.
  5. CRS recomendado: EPSG:32717 (UTM zona 17S para Ecuador).
👥 Gestión de usuarios (solo Admin)
  1. Ve a "Usuarios" en el menú.
  2. Puedes crear nuevos usuarios o eliminar existentes.
  3. La "Actividad" muestra todas las acciones realizadas.
Consejo: Presiona Enter dentro de cualquier campo para avanzar rápidamente al siguiente.
